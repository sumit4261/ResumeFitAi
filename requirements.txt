ResumeFit AI - Project Requirements
===================================

Runtime Requirements
- Node.js >= 18
- npm >= 9

Services Required
- MongoDB (local or cloud)
- Google GenAI API key

Backend Environment Variables (Backend/.env)
- MONGO_URI
- JWT_SECRET
- GOOGLE_GENAI_API_KEY

Default Local Ports
- Frontend: 5173
- Backend: 4040

Install Commands
- cd Backend && npm install
- cd Frontend && npm install

Run Commands
- cd Backend && npm run dev
- cd Frontend && npm run dev

Notes
- This repository is JavaScript/Node based.
- Python/pip requirements are not used in this project.
